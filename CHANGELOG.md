# Changelog

所有重要更改都会记录在此文件中。

## [0.3.0] - 2025-12-10

### 新增
- @vueuse/motion 声明式动画库
- @iconify/vue 图标库（20万+ 图标）
- @formkit/auto-animate 自动列表动画
- VueUse 实用组合式函数库
- 动画示例页面（Motion、列表动画、卡片悬停）
- 图标库示例页面（MDI、Heroicons、Lucide）
- VueUse Hooks 示例页面

### 改进
- 更新导航栏，添加新页面入口
- 完善 README 文档
- 更新 Kiro 开发规范

## [0.2.0] - 2025-12-10

### 新增
- 项目结构优化：composables、stores、types、views 目录
- Vue Router 路由系统
- Pinia 状态管理
- ESLint + Prettier 代码规范
- 路径别名 `@/`
- 前后端日志系统（tauri-plugin-log）
- GitHub Actions CI 检查工作流
- 完善的 README 文档

### 改进
- 主题选择器抽离为独立组件
- 添加 Tauri 功能演示页面

## [0.1.0] - 2025-12-10

### 新增
- 初始项目结构
- Tauri 2.x + Vue 3 + DaisyUI 5 + Tailwind CSS 4
- 32 个主题切换功能
- GitHub Actions 多平台打包
